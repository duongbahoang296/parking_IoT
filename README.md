# 🚗 Parking IoT System

## Giới thiệu
Hệ thống bãi đỗ xe thông minh (Parking IoT) sử dụng ESP32 và các cảm biến
để giám sát trạng thái chỗ đỗ xe theo thời gian thực.

## Công nghệ sử dụng
- ESP32
- Arduino IDE
- Web dashboard
- Sensor IR / Ultrasonic

## Chức năng
- Phát hiện xe ra / vào
- Hiển thị trạng thái bãi đỗ
- Gửi dữ liệu lên hệ thống web
- Cảnh báo khi bãi đầy
